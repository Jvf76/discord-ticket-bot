# 🎫 Discord Ticket Bot

Bot de sistema de tickets para Discord com gerenciamento de usuários e cargos, desenvolvido em **JavaScript com Node.js** utilizando **discord.js v14**.

---

## 🚀 Funcionalidades

* 📩 Criação de tickets
* 🔒 Canais privados para suporte
* 👥 Adicionar usuários ao ticket
* 🏷️ Adicionar cargos ao ticket
* ⚙️ Sistema simples e fácil de usar

---

## 🛠️ Tecnologias utilizadas

* JavaScript (Node.js)
* discord.js v14
* dotenv

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/Jvf76/discord-ticket-bot.git
```

Entre na pasta:

```bash
cd discord-ticket-bot
```

Instale as dependências:

```bash
npm install
```

---

## ⚙️ Configuração

Crie um arquivo `.env` baseado no `.env.example`:

```bash
cp .env.example .env
```

Depois preencha com suas informações:

```env
TOKEN=seu_token_aqui
CLIENT_ID=seu_client_id_aqui
GUILD_ID=seu_guild_id_aqui
```

---

## ▶️ Como executar

```bash
node index.js
```

---

## 📌 Observações

* Certifique-se de que o bot tem permissões para:

  * Criar canais
  * Gerenciar permissões
  * Enviar mensagens
